Python Data Analysis project as part of the LSE Data Analytics using Python. 

Business problem: NHS stakeholders disagree on whether expansion of current facilities was necessary or whether existing capacity could be put to better use. 

Project Objective: The purpose of this project is to analyse NHS appointment activity across service settings and seasonal periods in order to identify patterns in demand and healthcare capacity utilisation.

The analysis aims to determine whether existing healthcare capacity could be used more efficiently or whether expansion of facilities may be necessary to accommodate demand.

Data: the course provided 4 NHS datasets: appointments_regional.csv, national_categories.xlsx, actual_duration.csv, tweets.csv. These datasets provide information on appointment volumes, service categories and consultation durations across NHS service settings. The rest of the supporting data was obtained through web scraping the NHS official publications.
